# Spiderman-PCB
My first PCB is a compact NFC circuit with a 13.56 MHz chip, antenna coil, capacitor, resistor, and LED. It communicates wirelessly with NFC devices and lights when activated. I also added a Spider-Man design for creativity. This project taught me PCB layout, component placement, and trace routing.

**Steps**
1. Open your PCB design software (such as EasyEDA).
2. Create a new project and open the schematic editor.
3. Add the main NFC chip (13.56 MHz IC).
4. Add an NFC antenna coil component or design a custom coil.
5. Add supporting parts: capacitor, resistor, and LED.
6. Connect all components with wires according to the circuit diagram.
7. Check the schematic for missing or incorrect connections.
8. Convert the schematic into PCB layout mode.
9. Arrange the components neatly on the board.
10. Route copper traces between the parts and antenna.
11. Add board shape, labels, and artwork (like the Spider-Man design).
12. Run the design rule check (DRC) to find errors.
13. Preview the PCB in 2D/3D view.
14. Generate Gerber files for manufacturing.
15. Order the PCB or save the files for future use.
